# cobra-plus

Sistema de gestão de cobranças.

## stack

* react
* typescript
* vite
* tailwindcss

## funcionalidades (em progresso)

* clientes
* contratos
* parcelas
* dashboard
* cobranças do dia

## estrutura

```
src/
  components/
  pages/
  hooks/
  context/
  styles/
  types/
```

## rodando o projeto

```
npm install
npm run dev
```

## observações

Projeto em desenvolvimento. Algumas decisões de arquitetura e stack ainda podem mudar.
