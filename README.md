# cobra+

Sistema de gestão de cobranças.

## stack

* react
* typescript
* vite
* tailwindcss

## funcionalidades (em progresso)

- [x] estrutura inicial do frontend
- [x] layout base (sidebar + área principal)
- [x] módulo de dashboard criado
- [ ] cards de métricas
- [ ] lista de cobranças do dia
- [ ] gestão de clientes
- [ ] contratos e parcelas

## estrutura

src/
  app/
  modules/
    dashboard/
  shared/
    components/
      layout/
  assets/
  styles/
  types/

## rodando o projeto

```
npm install
npm run dev
```

## observações

Projeto em desenvolvimento. Algumas decisões de arquitetura e stack ainda podem mudar.
